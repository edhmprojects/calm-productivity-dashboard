# Calm Productivity Dashboard

A personal productivity dashboard designed to support calm, focused work.  
Built with HTML, CSS, and JavaScript as a learning project and a way to explore front‑end development patterns.

## Purpose

This project is a personal productivity dashboard built as a dual‑purpose learning environment:

- To develop and strengthen front‑end skills (HTML, CSS, JavaScript)
- To create a realistic, self‑contained application that can be used for practising automation testing with Playwright

The dashboard provides a safe, controlled space to explore:
- DOM interactions and UI behaviour
- Component‑based thinking
- Testability considerations
- Writing and maintaining automated tests as the project evolves

It is intentionally simple, calm, and modular so it can grow alongside my skills.

## Features (in progress)

- Modular layout for different widgets  
- Simple, clean UI  
- JavaScript‑driven interactions  

## Tech Stack

- **HTML** for structure  
- **CSS** for layout and styling  
- **JavaScript** for interactivity and logic  

No frameworks yet — intentionally lightweight while learning core skills.

## Roadmap

- Add initial components  
- Introduce modular JS structure  
- Explore turning the dashboard into a Progressive Web App (PWA)  
- Optional: experiment with React or a mobile‑app wrapper in future  

## Status

Early development. The core layout and structure are being established before adding interactive components.

The project is currently focused on:
- Building small, testable UI elements
- Establishing clean, readable JavaScript patterns
- Preparing the codebase for future Playwright automation practice

As the dashboard grows, it will serve as both a personal tool and a long‑term automation practice environment.
